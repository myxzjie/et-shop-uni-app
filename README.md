### 项目说明

1.  项目基于Vue-uniApp，使用colorUi开发. 感谢各位大佬～. 
2.  项目功能：
> * 自定义TabBar与顶部
> * banner轮播
> * 通知公告
> * 酒品分类
> * 发现/文章
> * 购物车
> * 个人中心
> * 促销功能(新品，推荐，热销、促销)
> * 签到
> * 团购
> * 秒杀
> * 砍价
> * 收藏
> * 积分
> * 优惠卷
> * 分销/佣金
> * 我的余额
> * 推广二维码
> * 小程序分享等等...
> 


## 安装教程
使用工具nodejs，vscode，微信开发工具。
```
npm install
```

```
npm run dev:mp-weixin
```

```
npm run build:mp-weixin
```
